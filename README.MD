カーソルフリーテキストエディタ (Windows)

Flutter/Dartで構築した、任意の位置に文字を打ち込めるテキストエディタです。
実装内容

    座標系

        画面上の行・列（Row/Col）と、実際の描画ピクセル座標を同期。

        テキストの増減に合わせて、配置済みの図形がずれないように追従させるロジックの実装。

    描画

        CustomPainterを使用。標準の入力フォームは使わず、一文字ずつの描画とカーソル位置を自前で制御。

        日本語入力（IME）の未確定文字がカーソル位置に正しく出るように調整。

    図形との共存

        テキストエディタの画面上に、直線、L型線、テーブル、矩形などの図形を配置。

        マウス操作による図形のリサイズや移動。

AI（Gemini 3）との進め方

AIの誤回答や修正による混乱を防ぐため、以下の手順で開発しました。

    方針の先行確定

        いきなりコードを書かせず、まず「修正方針」を出させて、それを見てから実装を指示。

    デバッグ手順

        修正が複雑になって迷走し始めたら、一度元のコードに戻し、変数のログを出させてから原因を特定。

    情報の渡し方

        AIが勝手にファイルを省略して読み飛ばさないよう、ファイル指定や全コードの貼り付けで情報を強制的に同期。
